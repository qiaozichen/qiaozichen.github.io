本项目演示了如何在 Hugo 静态站点中嵌入交互式图表。主要技术栈：

- 前端：Chart.js + Bootstrap
- 后端：Flask 提供 JSON API
- 部署：Docker + GitHub Actions

> 详细源码可在 GitHub 上获取：<https://github.com/qiaozichen/sample-dashboard>
